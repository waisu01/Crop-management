# 農作物管理システム

## 概要
このシステムでは、農作物の管理を行うことができます。

商品の新規登録から編集、削除を行うことができ、各商品の在庫を調整することができます。

## 主な機能
- ログイン、ログアウト、新規登録機能
- 商品一覧画面
- 商品新規登録、編集、削除機能
- 商品情報詳細画面
- 商品部分一致検索機能
- ページネーション機能

## ローカル開発環境
```
PHP 8.4.5　
MySQL 5.7.39　
Laravel 10.10　
```
